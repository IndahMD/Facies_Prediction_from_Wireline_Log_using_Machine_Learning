# Facies_Prediction_from_Wireline_Log_using_Machine_Learning
This repository contains code to predict Facies rock from wireline log

Folder structured:
* Data :
  - Contains wireline data in LAS format that we used to predict facies
  - Labeled facies of rock was given from 2 well log. Facies was labeled manually from geologist.
